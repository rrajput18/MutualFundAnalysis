# Mutual Fund Analytics Platform

An end-to-end data engineering and ETL pipeline for mutual fund data analysis, using Python, SQL, and SQLite.

## Project Structure

- `data/raw/` : Original datasets and live API fetches.
- `data/processed/` : Cleaned and normalized datasets.
- `sql/` : Database DDL schemas and analytical queries.
- `notebooks/` : Jupyter notebooks for exploratory data analysis (EDA).
- `dashboard/` : Power BI / Tableau dashboard files.
- `reports/` : Generated reports and presentations.
